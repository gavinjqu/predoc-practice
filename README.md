# predoc

Personal collection of materials for predoc preparation.

## Layout

- `predoc_data_task/` — sample data tasks (CPS, etc.) with full analyses in Quarto/Jupyter.
- `practice/` — practice problem sets and worked solutions.
- `ad_hoc_practice/` — one-off exercises, scratch work.
- `data/` — shared raw data files.
- `documents/` — references, notes, papers.
- `research.md` — running notes on programs, advisors, and applications.
- `pyproject.toml` / `uv.lock` — Python environment (managed with `uv`).

## Setup

```sh
uv sync
source .venv/bin/activate
```
